# HonoJS Stareter Kit

## Getting Started

First, run the development server:

```bash
pnpm dev
```

To build:

```bash
pnpm build
```

## Environment Variables

| Variable Name       | Variable Use                       | Sample Value                               |
| ------------------- | ---------------------------------- | ------------------------------------------ |
| PORT                | Port                               | 8080                                       |
| HOSTNAME            | Hostname                           | example.com                                |
